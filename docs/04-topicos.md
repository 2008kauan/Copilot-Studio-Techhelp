# Tópicos do Copilot

## Introdução

No Microsoft Copilot Studio, os **Topics** representam os assuntos que o copiloto é capaz de compreender e responder.

Cada tópico possui:

- Frases de gatilho (Trigger Phrases)
- Fluxo de conversa
- Respostas personalizadas
- Condições para encerramento ou encaminhamento

---

# Tópico 1 — Boas-vindas

## Objetivo

Recepcionar o usuário e iniciar a conversa.

### Trigger Phrases

- Olá
- Oi
- Bom dia
- Boa tarde
- Boa noite

### Resposta

> Olá! 
>
> Eu sou o **TechHelp Copilot**.
>
> Estou pronto para ajudar você com dúvidas técnicas.
>
> Como posso ajudar hoje?

---

# Tópico 2 — Problemas com Internet

## Trigger Phrases

- Minha internet caiu
- Sem conexão
- Wi-Fi não funciona
- Internet lenta

### Fluxo

1. Confirmar o problema.
2. Solicitar informações adicionais.
3. Sugerir verificações básicas.
4. Confirmar se o problema foi resolvido.

---

# Tópico 3 — Problemas com Computador

## Trigger Phrases

- Computador travando
- Notebook lento
- Tela azul
- PC não liga

### Fluxo

- Identificar o sintoma.
- Sugerir procedimentos básicos.
- Caso necessário, encaminhar para atendimento humano.

---

# Tópico 4 — Instalação de Programas

## Trigger Phrases

- Instalar Python
- Instalar Git
- Instalar VS Code

### Fluxo

- Identificar o software.
- Apresentar o passo a passo.
- Confirmar sucesso da instalação.

---

# Tópico 5 — Atendimento Humano

## Trigger Phrases

- Quero falar com um atendente
- Atendimento humano
- Suporte técnico

### Fluxo

- Registrar a solicitação.
- Informar que o atendimento será transferido.
- Encerrar a interação.

---

# Considerações

A divisão por tópicos torna o copiloto mais organizado, facilita a manutenção e permite a expansão futura da solução.
