## Sobre o projeto

O **TechHelp Copilot** é um assistente virtual projetado para oferecer suporte técnico inicial aos usuários, respondendo dúvidas frequentes, orientando na resolução de problemas e encaminhando solicitações para atendimento humano quando necessário.

Este repositório documenta toda a arquitetura e o planejamento do copiloto com base nos recursos do Microsoft Copilot Studio.

---

## Objetivos

- Criar um fluxo de conversa personalizado.
- Definir tópicos de atendimento.
- Planejar respostas utilizando IA Generativa.
- Personalizar mensagens de erro.
- Documentar a arquitetura da solução.

## Observação

Este projeto documenta o planejamento de um copiloto baseado no Microsoft Copilot Studio.

A implementação prática não foi realizada porque, durante o desenvolvimento, o acesso à plataforma exigia uma conta Microsoft corporativa ou educacional, indisponível para este ambiente. Toda a documentação foi elaborada para representar fielmente como a solução seria implementada.
